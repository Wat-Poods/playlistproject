This is the Playlist Project

The Playlist Project is a list of themed playlists, shared by their creators for the enjoyment of anyone interested. Each playlist has been curated around a theme, with the length being around 60 minutes. Each month there is a new theme. To contribute, you can create a pull request. If you are unfamiliar with GitHub and would still like to contribute, you can submit a link to pat@patwoods.co. Email submissions will be updated as time allows.
